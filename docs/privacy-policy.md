# Privacy Policy

Last updated: 2026-06-23

TabScout is a local-first Chrome extension for searching, switching, and managing Chrome tabs, history, bookmarks, and tab groups.

## Data The Extension Accesses

TabScout may access:

- Open tab titles, URLs, favicons, pinned state, window IDs, tab indexes, and tab group metadata.
- Chrome history titles and URLs, only after the user grants the optional History permission.
- Chrome bookmark titles, URLs, and folder paths, only after the user grants the optional Bookmarks permission.
- Local extension settings such as theme preference, recent tab activity, and undo metadata.

## How Data Is Used

The extension uses this data only to provide local command-panel features:

- Search and rank tabs, history, bookmarks, and tab groups.
- Switch to an open tab and focus its Chrome window.
- Open a history or bookmark result.
- Close selected, duplicate, same-domain, or matching tab results.
- Restore recently closed tabs when undo metadata is available.
- Display local settings and permission states.

## Local Processing

All search, scoring, ranking, settings, and undo metadata are processed locally in Chrome. The extension does not upload browser data to external servers.

## Data Sharing

TabScout does not sell, transfer, or share user data with third parties.

## Analytics, AI, And External Services

TabScout does not include analytics SDKs, tracking services, AI services, cloud sync, or external service calls.

## Webpage Content

TabScout does not index webpage body content. It does not inject content scripts into webpages and does not request host permissions.

## Optional Permissions

History and Bookmarks permissions are optional. The extension asks for them in context when the user opens or searches the corresponding scope. If the user does not grant an optional permission, the related search scope remains unavailable while the rest of the extension continues to work.

## Local Data

The extension stores local settings such as theme preference, recent tab activity, and undo metadata in Chrome extension storage. Users can clear local extension data from the settings page.
